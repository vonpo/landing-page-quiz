# manual.co challenge landing page

# tech stack
* React (hooks, context API)
* Typescript
* Less
* MaterialUI (css-hook-api)
* Tests(jest)
* webpack

## Browser

Tested on:
* chrome (windows)
* firefox (windows)
* edge (windows)
* chrome (android)
* safari (ipad ios)

for me.
